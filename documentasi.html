AI Setup Guide

Setup Marketing AI Agent

Panduan instalasi lengkap untuk menjalankan AI Agent berbasis DeepSeek dan Marketing Skills framework.

1

Persiapan Repositori

Buka terminal atau CMD, lalu klon proyek dan masuk ke foldernya:

git clone https://github.com/username-kamu/nama-repo-kamu.git
cd skillaichatbot
2

Virtual Environment

Gunakan venv agar library tidak bentrok dengan sistem.

Windows 
python -m venv venv
venv\Scripts\activate
Linux (Debian/Ubuntu) 
python3 -m venv venv
source venv/bin/activate
3

Instalasi Requirements

Instal semua library yang dibutuhkan dari file requirements.txt:

pip install -r requirements.txt
4

Uji Coba Aplikasi

Jalankan streamlit untuk melihat interface lokal:

streamlit run main_1.py

Akses di http://localhost:8501. Masukkan API Key DeepSeek pada sidebar.

5

Deployment (PM2)

Jalankan di background (VPs) menggunakan PM2:

deactivate
pm2 start "streamlit run main_1.py --server.port 8501" --name marketing-ai
pm2 status

Cek status bot.

pm2 logs marketing-ai

Cek log error.

pm2 restart marketing-ai

Restart setelah update kode.

Tips Keamanan (Opsional)

Simpan API Key di file .env agar lebih praktis dan aman:

DEEPSEEK_API_KEY=sk-xxxxxx...
