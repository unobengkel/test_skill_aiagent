📋 Prasyarat

  • Python 3.8 atau lebih baru.
  • DeepSeek API Key.
  • PM2 (Opsional untuk VPS).

🚀 Panduan Instalasi

1

Klon Repositori

Ambil source code dari repositori Github:

git clone https://github.com/unobengkel/test_skill_aiagent.git
cd test_skill_aiagent
2

Virtual Environment

Isolasi library agar tidak bentrok dengan sistem utama:

Windows 
python -m venv venv
venv\Scripts\activate
Linux / macOS 
python3 -m venv venv
source venv/bin/activate
3

Instal Dependensi

pip install -r requirements.txt
4

Uji Coba Lokal

Jalankan dengan port dan alamat spesifik:

streamlit run main.py --server.port 9000 --server.address 0.0.0.0

Buka browser di http://localhost:9000. Masukkan API Key DeepSeek pada sidebar.

5

⚙️ Deployment (Production)

Gunakan PM2 untuk menjaga aplikasi tetap berjalan di background:

deactivate
pm2 start "streamlit run main.py --server.port 9000 --server.address 0.0.0.0" --name marketing-ai

Cek status bot dengan perintah pm2 status.
